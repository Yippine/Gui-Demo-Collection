# Gui Demo Collection

要運行這個程序，請按照以下步驟操作：

1. 打開命令提示符或終端，導航到 `1-file_processor` 文件夾（包含 `setup.py` 的文件夾）。

2. 運行以下命令來安裝程序：
   ```
   pip install -e .
   ```
   這會在「可編輯」模式下安裝程序，這意味著你可以修改源代碼，而不需要重新安裝。

3. 現在你可以通過以下方式運行程序：
   ```
   python -m file_processor.main
   ```
   或者，如果你希望直接運行：
   ```
   file_processor
   ```
